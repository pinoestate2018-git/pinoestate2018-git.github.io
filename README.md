- Google Fonts

## GitHub Pagesへのアップロード手順

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイル一式をリポジトリ直下にアップロードします。
3. GitHubのリポジトリ画面で `Settings` を開きます。
4. 左メニューの `Pages` を開きます。
5. `Build and deployment` の `Source` を `Deploy from a branch` にします。
6. `Branch` を `main`、フォルダを `/root` にして `Save` を押します。
7. 数分後、GitHub Pagesの公開URLが表示されます。

## 更新方法

文章を変更する場合は各HTMLファイルを編集します。色や余白などのデザイン調整は `css/style.css`、スクロールアニメーションなどの動きは `js/script.js` を編集します。
